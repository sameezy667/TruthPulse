## Key Architectural Decisions

### 1. Discriminated Union
**Why:** Enables type-safe routing in GenerativeRenderer
**Benefit:** TypeScript knows exact shape of data based on `type` field

### 2. DeepPartial
**Why:** Components must handle incomplete streaming data
**Benefit:** No crashes during streaming, graceful degradation

### 3. Vercel AI SDK
**Why:** Purpose-built for streaming AI responses
**Benefit:** Handles complexity of streaming, validation, error handling

### 4. Zod Schema
**Why:** Runtime validation + TypeScript types from single source
**Benefit:** Type safety + runtime safety, no drift

### 5. Component-per-Type
**Why:** Each response type has unique UI requirements
**Benefit:** Clean separation, easy to maintain, no conditional mess
